# functions
関数を定義する
- `simpleFunctions.ipynb`: 引数の無い関数
- `argumentsTest.ipynb`: 引数のある関数
- `factorial.ipynb`: 再帰関数の例: 階乗
- `isPrime.ipynb`: 戻り値としてタプルを使う例: 素数判定
- `stat0.ipynb`: 戻り値としてタプルを使う例: 簡単な統計
- `scopeTest.ipynb`: 関数とスコープ
- `argumentTest1.ipynb`: 引数の明示と省略
- `quiz.ipynb`: 確認テスト 
- `exercise.ipynb`: 演習問題
